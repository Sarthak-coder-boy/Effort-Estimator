Formula :    Effort = a1 (KLOC) a2 * PM

a1  and  a2  =  constants , which depends on category of software project

  • Organic :    a1 = 2.4   and    a2 = 1.05
  • Semi Detached :    a1 = 3.0   and    a2 = 1.12
  • Embedded :    a1 = 3.6   and    a2 = 1.20

KLOC    =   Size of lines in kilo lines of code

PM    =   Person-Months , it is the unit of effort put in the development of project
